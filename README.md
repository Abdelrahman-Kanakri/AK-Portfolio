# Abdelrahman Kanakri — Portfolio

Personal portfolio website showcasing my work as a **Data Scientist & AI Engineer**.

🔗 **Live Site**: _coming soon_

---

## ✨ Features

- **Responsive Design** — Fully optimized for desktop, tablet, and mobile
- **Smooth Animations** — Scroll-triggered reveal animations using IntersectionObserver
- **Project Filtering** — Filter projects by category (Deep Learning, ML, NLP, AI & RAG)
- **Interactive Timeline** — Visual work experience timeline with glassmorphism cards
- **Contact Form** — Integrated with [Formspree](https://formspree.io) for email delivery
- **Dark Theme** — Modern dark UI with gradient accents and glass-card effects

---

## 🛠️ Tech Stack

| Layer     | Technology                                                                 |

## 🚀 Getting Started

No build tools or dependencies required — just open the site in a browser.

```bash
# Clone the repo
git clone https://github.com/Abdelrahman-Kanakri/AK-Portfolio.git

# Open directly
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```

Or use any local dev server:

```bash
# Python
python -m http.server 3000

# Node (npx)
npx serve .
```
